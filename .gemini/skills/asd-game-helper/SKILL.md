# ASD/LDゲーム開発ヘルパー

このスキルはASD/LD向けゲーム開発を支援します。

## 機能

1. **コードレビュー**: ASD/LD配慮の観点からコードをレビュー
2. **UI/UX提案**: 視覚刺激軽減、優しいメッセージの提案
3. **アクセシビリティチェック**: WCAG 2.2 AA準拠の確認

## 使用例

```
このゲームのUIがASD/LDの子供に適しているかレビューして
```

## チェックポイント

### 視覚的配慮
- [ ] 色のコントラスト比 4.5:1 以上
- [ ] アニメーションは控えめ（30fps以下推奨）
- [ ] 点滅する要素がない
- [ ] 穏やかな色使い（パステル系）

### テキスト配慮
- [ ] ひらがな中心
- [ ] 短い文章
- [ ] 優しい言葉遣い
- [ ] 否定的表現を避ける

### 操作性
- [ ] タッチ/クリック両対応
- [ ] 十分なタップ領域（48px以上）
- [ ] フォーカス可視性
- [ ] キーボード操作対応
